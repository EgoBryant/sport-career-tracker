# 🏆 Sport Career Tracker (FIFA 23 Style)

> **"Become a Legend."** Личный интерактивный веб-трекер спортивных достижений, стилизованный под режим карьеры и карточки Ultimate Team из игры FIFA 23. 

Проект создан для геймификации тренировочного процесса, отслеживания прогресса в реальном времени и глубокой аналитики физической активности.

---

## ⚽ Текущий модуль: Футбол (MVP)

Проект начинается с автоматизации футбольной статистики. Моя позиция — **Нападающий**. Расписание: матчи по понедельникам и средам (90 минут + разминка).

### 📊 Что трекается внутри системы:
*   **Регулярный сезон:** Учет исходов матчей (Победа / Ничья / Поражение), статистика забитых голов и голевых передач.
*   **Кубок (Плей-офф):** Отдельная статистика для внутренних турниров, проводимых в конце игрового дня.
*   **Интеграция со смарт-часами:** Импорт физических показателей за матч (время разминки/игры, общий пробег в км, сожженные калории, средний и максимальный пульс).

### 🎮 Игровые механики (FIFA 23 Core):
1.  **Динамическая карточка игрока:** Изменение общих характеристик (скорость, удар, пас, физика) на основе реальных данных с матчей и часов.
2.  **Рыночная стоимость:** Трансферная стоимость, которая растет от забитых голов, побед и индивидуальных наград, или падает при плохой форме.
3.  **Индивидуальные награды:** Алгоритмическое присуждение званий *Игрок матча*, *Команда недели*, *Игрок месяца* и *Игрок года*.
4.  **Модуль тренировок:** Логирование домашних и полевых тренировок для прокачки выносливости и физической формы.

## 📈 Дорожная карта проекта (Roadmap)

- [ ] **Phase 1: Football MVP (Текущий этап)**
  - Проектирование БД и согласование формул расчета стоимости/рейтинга.
  - Разработка бэкенда на сохранение матчей и тренировок.
  - Фронтенд в темных тонах FIFA 23 UI с интерактивной карточкой.
- [ ] **Phase 2: Расширение спортивной экосистемы**
  - Модуль **Плавание**: Трекинг дистанций, стилей, времени на 50/100 метров.
  - Модуль **Домашние тренировки**: Счётчик подтягиваний, отжиманий, прогресс в весах и выносливости.
- [ ] **Phase 3: Аналитика и Сводные отчеты**
  - Графики формы за месяц/год.
  - Сравнение показателей по неделям.

*   **Backend:** Python / C# (в процессе выбора)
